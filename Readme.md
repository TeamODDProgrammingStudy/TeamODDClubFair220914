### 동아리 박람회 결산

- 박람회 참여 인원이 생각보다 많았다.
    - 코로나 상황임에도 100~150명 방문
    - 참여상 100세트 전량 소진
    - 난이도 보통 상품(페레로로쉐)
        - 시작 후 1시간 30분 경 전량 소진
    - 난이도 어려움 상품(문화상품권 5000원)
        - 시작 후 2시간 경 전량 소진
    - 난이도 아주 어려움(고속충전기 10000원)
        - 2개 남음
- 다만 사람들도 즐겁게 참여하신 것 같다.
- 후반부에 상품이 떨어져 받지 못하더라도 참여하시는 분들도 많았다.
